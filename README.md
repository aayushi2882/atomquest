# AtomQuest — Enterprise Goal Intelligence Platform

AtomQuest is a modern enterprise-grade goal management and approval platform built with React and Tailwind CSS.

It enables Employees, Managers, and Admins to collaborate through a structured quarterly goal workflow system with role-based dashboards and premium UI interactions.

---

## Features

* Role-based Authentication
* Employee Goal Creation Workflow
* Manager Approval System
* Admin Analytics Dashboard
* Premium Dark SaaS UI
* Responsive Dashboard System
* Goal Lifecycle Tracking
* Activity Monitoring
* LocalStorage-based Persistence
* Framer Motion Animations

---

## Tech Stack

## User Roles

### Employee

* Create goals
* Submit goals for approval
* Track progress

### Manager

* Review employee submissions
* Approve/reject goals
* Monitor team workflow

### Admin

* Organization-wide visibility
* Analytics monitoring
* Employee activity overview

---

## Workflow

Employee → Creates Goals
↓
Manager → Reviews & Approves
↓
Admin → Monitors Organization Activity
